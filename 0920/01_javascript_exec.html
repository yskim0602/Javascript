01_자바스크립트 활용예제

자바스크립트 언어를 활용하여 다양한 콘텐츠를 만들어 본다.

1. 전구이미지를 활용하여 버튼을 클릭하면 이미지의 'src'속성값을 변경하여 다른 이미지가 나오게 하기


2. 버튼을 클릭하면 해당하는 요소에 글자의 크기를 변경하기(확대, 축소)

건강보험공단, 산업인력공단 등 사이트에서 사용하는 글자확대, 축소하기

  • onclick : 자바스크립트 이벤트의 종류로서 클릭을 했을때 이벤트가 발생되어 실행됨.
  • document : 문서객체를 말하며, html문서를 의미하기도 함
  • getElementById : 사용자가 선택하고자 하는 아이디명을 적을때 사용함 일종의 css의 #id선택자와 같음
  • style : 선택한 요소에 css서식을 적용하고자 할 때 사용.
  • fontSize : 글자크기를 변경하고자 하는 자바스크립트에서 사용하는 스타일 속성임.

3. 모바일 화면에서 버튼을 클릭시 조명끄기, 조명켜기 기능 구현하기
